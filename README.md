# DiceCream.DCorp

## Description
A C# project for managing the operations of DiceCream Corporation.

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/Alex2119/DiceCream.DCorp.git
   ```
   
Navigate to the project directory:
cd DiceCream.DCorp
